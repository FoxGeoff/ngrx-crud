## Project Introduction

### Task: Add bootstrap

1. `npm i bootstrap`
2. Configure Bootstrap CSS and JS files in 'angular.json'

```json
...
            "assets": [
              "src/favicon.ico",
              "src/assets"
            ],
            "styles": [
              "src/styles.css",
              "node_modules/bootstrap/dist/css/bootstrap.min.css"
            ],
            "scripts": ["node_modules/bootstrap/dist/js/bootstrap.min.js"]
          },
...
```
